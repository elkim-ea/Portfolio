backend/src/main/java/com/greenceipt/backend
 ├─ config/       # 프로젝트 전역 설정 (CORS, JWT, Swagger 등)
 ├─ controller/   # REST API 엔드포인트
 ├─ domain/       # 엔티티 클래스 (DB 테이블 매핑)
 ├─ dto/          # 데이터 전송 객체
 ├─ exception/    # 예외 처리 클래스
 ├─ repository/   # JPA Repository 인터페이스
 ├─ service/      # 비즈니스 로직 (Service, ServiceImpl)
 ├─ util/         # 공통 유틸리티(함수)
 └─ BackendApplication.java  # Spring Boot 메인 실행 파일
